PNNL computing landscape

This directory includes various hacks of D3 examples to illustrate PNNL computing capabilities. Unless otherwise noted, sizes of visualization elements represent overall PNNL strength/capabilities in an area.

Caveats

  • Visualizing [on the web](http://sobolevnrm.github.io/Computing) seems to work OK. However, visualizing locally can be a problem: this has been tested in Safari 8 (in developer mode) on Mac OS 10.10.5 -- if you want to use other browsers, please follow the instructions for bypassing CORS errors.
